# Diagnostic Page Layout Reference (from Miro)

## Layout Structure

The diagnostic page should follow this structure:

### Left Column (40% width)
- **Title**: "Diagnóstico"
- **Content Blocks**: 4 placeholder/content sections (gray boxes in mockup)

### Center/Right Area (60% width)
- **Plant Visual**: Large plant image centered at top
- **Agency Info Section**: Below plant
  - Agency Name (heading)
  - Health bar (green)
  - Size bar (blue)

## Key Changes from Current Layout
1. Plant moves from far right to center-right
2. Left sidebar with "Diagnóstico" title and content blocks
3. Agency name, health bar, and size bar move below the plant (instead of left side)
4. More balanced, centered layout

## Visual Hierarchy
- Plant is the focal point (center-top)
- Left navigation/content is secondary
- Agency metrics are below plant for context
